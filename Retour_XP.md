# Retour d'expérience : 

Première API que je fais avec AdonisJS 6, et franchement c’était une bonne surprise.
La création de fichiers via les commandes ace est super pratique, tout est bien organisé, et la mise en place des middlewares ou des tests se fait presque toute seule.

J’ai essayé d’ajouter Better Auth au début, mais comme y’a pas encore de module vraiment adapté à Adonis, ça m’a un peu ralenti. Du coup je suis revenu sur le système d’auth intégré, comme recommandé dans la doc et au final ça fonctionne super bien. 

Pour le front, j'ai réalisé un design extrêment basic. 
Le projet étant surtout là pour tester la stack, j’ai préféré aller au plus simple côté interface.
C’est pour ça que j’ai utilisé Material UI : rapide à mettre en place, propre, et suffisant pour ce type de projet.

J'ai également utiliser docker pour m'assurer que le projet fonctionne correctement chez tout le monde.
